## 部署方法（3 分钟）

### 第 1 步：Fork 本仓库

点右上角 `Fork` 按钮。

### 第 2 步：注册 PushPlus（免费）

1. 打开 https://www.pushplus.plus/ → 微信扫码登录
2. 在「一对一推送」页面复制你的 `Token`
3. **粘贴好备用**

### 第 3 步：配置 GitHub Secrets

进你 Fork 后的仓库 → `Settings` → `Secrets and variables` → `Actions` → `New repository secret`

| Secret 名称 | 值 |
|------------|---|
| `PUSHPLUS_TOKEN` | 第 2 步复制的 PushPlus Token |

### 第 4 步：启用 Actions

点 `Actions` 标签 → `I understand my workflows, go ahead and enable them`

### 第 5 步：手动测试

`Actions` → `每日选股扫描` → `Run workflow` → `Run workflow`

等 1 分钟 → 微信收到报告，搞定 ✅

之后每个**交易日早上 9:00（北京时间）**自动跑，报告自动推到你微信。

**Q: 要钱吗？**

**Q: 我电脑要一直开着吗？**

**Q: 可以换港股/美股吗？**
